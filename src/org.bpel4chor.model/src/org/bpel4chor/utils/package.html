Package Specification

This package contains classes for reading and writing BPEL4Chor artifacts(pbd, topology, grounding), the Constants class, Builder class etc. can also be found here.

BPEL4ChorReader and BPELChorWriter

The BPEL4ChorReader for now just reads in the BPEL files and translate it into a BPEL process.
The BPEL4ChorWriter does vice versus, and further more it writes the topology and grounding into file via. StAX.

BPEL4ChorFactory

BPEL4ChorFactory provides the method to create all artifacts for topology and grounding.

BPEL4ChorConstants

BPEL4ChorConstants holds all the constants, for instance the ABSTRACT_PROCESS_NAMESPACE
